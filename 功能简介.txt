================================================================================
                    掏鸟 (TaoNiao) — 开放世界PVP助手 v0.3.0
================================================================================

掏鸟是一款专为魔兽世界经典怀旧服（永久60级）打造的开放世界野外PVP全能助手。
集敌人雷达、威胁评估、仇人管理、队伍协调、位面切换、战斗统计于一体，
为野外PVP玩家提供全方位的战场态势感知。

================================================================================
一、敌人侦测与追踪
================================================================================

  ● 多源实时扫描：战斗日志 / 目标 / 鼠标指向 / 姓名板 / 焦点 / 可视单位
  ● 显示敌人姓名（跨服去后缀）、职业、等级、公会、最近发现时间
  ● 潜行侦测：自动识别潜行/消失/影遁技能，触发警报
  ● 骷髅级敌人等级推算
  ● 敌人60秒过期自动清理
  ● 与 Spy 插件互通数据，丰富敌人信息

================================================================================
二、危险指数评估
================================================================================

  多因素加权评分（0-99），综合评估当前威胁等级：
  ● 附近敌人数量与职业权重（战士 > 盗贼 > 法师 > ...）
  ● 职业克制矩阵：克你的敌人加分，你克的敌人减分
  ● 个人战绩：你对某敌人的历史胜率
  ● 公会战绩：你对敌方公会的整体胜率
  ● 友方减伤：附近队友和非组队友方减少威胁值

  威胁分级：
    80+  致命    60+  高危    40+  危险
    25+  警觉    10+  平静    <10  和平

  ● 迷你HUD上的"呼吸灯"动画反映当前威胁等级

================================================================================
三、见之必杀 (KOS) — 死刑名单
================================================================================

  ● 永久保存你的宿敌名单（按角色存储）
  ● 右键敌人列表即可加入，附带"罪名"说明
  ● KOS 敌人在列表中置顶，显示红色骷髅标记
  ● 支持搜索、排序（按名称/胜/负/时间）、编辑、删除
  ● 发现 KOS 目标时自动弹窗 + 音效提醒

================================================================================
四、喊话通报系统
================================================================================

  ● 手动一键通报：队伍 / 团队 / 公会 / 本地防务
  ● 自动通报：发现新敌人时自动喊话（可关闭）
  ● KOS 专属通报："这人我见之必杀！"
  ● 潜行警告通报
  ● 通报冷却可调（0-120秒）
  ● 飞行路径自动静默
  ● 可设置仅通报 KOS 目标

================================================================================
五、团队助手
================================================================================

  ● 密语自动组队：关键字"加 1 进组"自动邀请
  ● 自动转团：5人满时自动转为团队
  ● 自动提升：新成员自动提升为助理
  ● 自动自由拾取
  ● 队伍保存/恢复：一键保存当前队伍，随时一键恢复
  ● 解散队伍：含告别语，权限检查
  ● 批量提升/降职
  ● HUD 面板上可自定义最多5个快捷操作按钮

================================================================================
六、位面助手
================================================================================

  ● 解析生物GUID识别当前位面
  ● 与 NovaWorldBuffs 联动获取准确位面信息
  ● 配置最多5个位面切换帮手（密语切换位面）
  ● 位面切换快捷菜单（组队中自动禁用防误操作）
  ● 位面配置分享：生成/导入编码字符串

================================================================================
七、战斗统计
================================================================================

  ● 今日击杀 / 死亡 / 荣誉击杀 / 胜率
  ● 总击杀 / 总死亡
  ● 逐对手战绩：对每个人的胜负场次、最近遭遇、区域
  ● 公会战绩（影响危险指数评分）
  ● 战斗日志（最多500条，按日期筛选）
  ● 击杀TOP10与被杀TOP10排行榜
  ● 历史对手列表（搜索、排序）

================================================================================
八、弹窗通知与音效
================================================================================

  ● 浮动弹窗通知（最多5行），可拖动定位
    - 潜行敌人出现
    - KOS 目标发现
    - 队友击杀 / 队友阵亡
    优先级：宿敌 > 潜行 > 队友阵亡 > 队友击杀
  ● 音效警示（可自定义声道：SFX/主音量/音乐等）
    - detected-kos.mp3     (KOS目标)
    - detected-stealth.mp3 (潜行敌人)
    - detected-nearby.mp3  (附近敌人)
    - list-add.mp3 / list-remove.mp3
  ● 可设置"仅高危警报"模式（仅宿敌+潜行触发声音）

================================================================================
九、界面与操作
================================================================================

  ● 常驻HUD面板：位置、坐标、统计、快捷按钮、敌人列表
  ● 迷你折叠模式：精简图标+呼吸灯
  ● 详细窗口 (980x860)：侧边栏导航（数据/KOS/位面/团队/设置）
  ● 右键菜单：添加KOS、快速通报
  ● 面板透明度/缩放可调，4套配色方案
  ● 副本/战场内自动禁用
  ● 所有面板可解锁拖动定位

================================================================================
十、斜杠命令
================================================================================

  /tn lock      锁定/解锁HUD（解锁后可拖动）
  /tn reset     重置HUD位置
  /tn clear     清空附近敌人列表
  /tn sound     开关音效
  /tn sound kos 开关"仅高危警报"模式
